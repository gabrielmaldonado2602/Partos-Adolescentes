# Análisis de Partos Adolescentes en República Dominicana

### Descripción General del Proyecto

Este proyecto tiene como objetivo proveer insights para el área de salud y contribuir a la concientización sobre una situación que ha estado afectando este país en los últimos años. A partir del análisis de distintos aspectos de datos sobre partos adolescentes, se busca identificar tendencias, hacer recomendaciones informadas, y obtener una comprensión más profunda sobre esta problemática.

### Herramientas

- Python: Limpieza de Datos 
- MySQL Workbench: Extracción de Información
- Power BI: Creación del Dashboard
